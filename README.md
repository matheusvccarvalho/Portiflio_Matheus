## 📂 Projetos

### 🔄 Conversor de Arquivos Multifuncional

Sistema inspirado em ferramentas como SmallPDF, com diversas funcionalidades:

* Conversão de arquivos (PDF, Word, Excel, ZIP)
* Visualização da primeira página de PDFs
* União de arquivos PDF
* Interface moderna e intuitiva
* Temas personalizados (Agro Verde 🌱, Clássico Azul 🔵, Moderno Escuro 🌑)

Sistema voltado para cálculo de orçamentos utilizando padrões como Cilia e Audatex:

* Histórico de orçamentos
* Destaque visual de valores
